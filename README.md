# Sales Performance Dashboard (Power BI + SQL)

## Overview

This project presents an interactive **Sales Performance Dashboard** built using Power BI with **SQL Server as the data source**.
The solution demonstrates end-to-end data handling—from data storage and querying to visualization and insights.

---

## Data Source

* Imported Superstore dataset (CSV) into **SQL Server**
* Created database and tables for structured storage
* Used SQL queries for initial data exploration and validation

---

## Key Objectives

* Track **sales performance vs target**
* Analyze **profitability across regions**
* Identify **top and bottom performing sub-categories**
* Evaluate **category efficiency (Sales vs Profit %)**
* Monitor **monthly sales trends with rolling averages**

---

## Dashboard Highlights

### 1. YTD Sales Performance

* Displays sales vs target with variance %
* Helps assess progress toward business goals

### 2. Regional Profit Margin vs Target

* Compares region-wise performance against benchmark
* Highlights underperforming regions

### 3. Top & Bottom Sub-categories

* Identifies best and worst performers based on Profit %

### 4. Category Efficiency Matrix

* Scatter plot (Sales vs Profit %) with quadrant analysis
* Highlights:

  * High-performing categories
  * Risk areas (high sales, low profit)
  * Growth opportunities

### 5. Monthly Sales Trend

* Time-series analysis with rolling average
* Identifies seasonality and trends

### 6. Key Insights Panel

* Dynamically generated insights using DAX

---

## Tools & Technologies

* **SQL Server** (Data storage & querying)
* **Power BI** (Visualization)
* **DAX** (Calculations & KPIs)
* **Power Query** (Data transformation)

---

## Key Learnings

* Designed a simple **data pipeline from SQL Server to Power BI**
* Practiced **SQL querying for data validation and exploration**
* Implemented **dynamic KPIs and ranking logic using DAX**
* Applied **time intelligence (YTD, rolling averages)**
* Built **interactive and insight-driven dashboards**
